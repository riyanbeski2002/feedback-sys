# Brand Alignment Notes for Ziptrrip Reference Screenshots

Once Ziptrrip dashboard screenshots are available, update the design system and implementation using the following checklist:

## Visual Checklist
- Capture typography family and font weights
- Measure heading sizes and body text scale
- Identify primary action color and hover treatment
- Note table row density and border style
- Note card radius, shadow depth and spacing rhythm
- Identify icon style and navigation treatment
- Match button height and input styling

## Implementation Checklist
- Update Tailwind theme tokens
- Update button, badge, card and table component styles
- Adjust spacing and section layout to align with the source design
- Replace placeholder icons or graphs if the reference UI uses a distinct pattern

## Figma Checklist
- Update component library based on screenshot references
- Keep the feedback flow visually native to the existing dashboard ecosystem

## Important Principle
Do not merely copy decorative styling. The goal is to make the new feature look like a natural extension of the Ziptrrip product rather than a standalone demo bolted onto it.
