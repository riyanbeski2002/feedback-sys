# MVP Scope and Demo Flow

## 1. MVP Objective
Show the end-to-end feedback intelligence loop in a working demo with realistic data and immediate UI updates.

## 2. MVP Inclusions
- Sample hotels
- Sample bookings
- Checkout simulation
- Notification preview page
- Detailed feedback form
- Weighted score calculation
- Hotel score aggregate update
- Listing page reordering based on score
- Admin dashboard updates
- Optional comment sentiment tagging

## 3. MVP Exclusions
- Real login or user accounts
- Production-grade notification sending
- Queue infrastructure
- Vendor admin workflows beyond flags and dashboard signals

## 4. Happy Path Demo
1. Open booking simulation page
2. Choose a booking with checkout due or completed
3. Click `Simulate Checkout`
4. Open notifications preview and show the channel designs
5. Click the linked detailed feedback flow
6. Fill ratings and optional comment
7. Submit form
8. Open hotel listing page and show score update and rank change
9. Open admin dashboard and show updated metrics and feedback insight

## 5. Demo Storyline Tip
Pick a hotel that is near a threshold so the score visibly changes enough to make the ranking impact obvious during demo.
