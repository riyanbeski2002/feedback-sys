```tsx
    <span className="text-[9px] font-bold text-muted-foreground uppercase tracking-widest">Navigation</span>
  </div>
  <SidebarMenu className="gap-0.5">
    ...
  </SidebarMenu>
</SidebarContent>
```

**Change 2 — Tighter nav item height and spacing.**

Update `SidebarMenu` gap from `gap-1` to `gap-0.5`.

Update `SidebarMenuButton` className: change `h-11` to `h-9`. Keep all existing active/inactive color logic unchanged.

Update the Link icon size: `size-5` → `size-4` (matches the mockup's smaller icon density at compact height).

Update the Link className gap: `gap-3` → `gap-2.5`.

Update the span font: `font-semibold` → `font-medium text-sm`.

**Change 3 — Tighter footer card padding.**

In `SidebarFooter`, change the outer div from `p-4` to `p-3`. Change the inner card div from `rounded-xl p-4` to `rounded-lg p-3`. This reduces the visual weight of the footer card to match B2B density.

The `collapsible="icon"` prop on `<Sidebar>` must remain unchanged. Do NOT touch `src/components/ui/sidebar.tsx` — all changes are className overrides in this consumer file only.

Full updated SidebarContent block target:
```tsx
<SidebarContent className="px-2 pt-2">
  <div className="px-2 mb-1 group-data-[collapsible=icon]:hidden">
    <span className="text-[9px] font-bold text-muted-foreground uppercase tracking-widest">Navigation</span>
  </div>
  <SidebarMenu className="gap-0.5">
    {data.navMain.map((item) => (
      <SidebarMenuItem key={item.title}>
        <SidebarMenuButton
          asChild
          isActive={pathname === item.url}
          tooltip={item.title}
          className={`transition-all duration-150 h-9 ${
            pathname === item.url
              ? "bg-primary/10 text-primary hover:bg-primary/15 hover:text-primary"
              : "text-muted-foreground hover:bg-muted hover:text-foreground"
          }`}
        >
          <Link href={item.url} className="flex items-center gap-2.5">
            <item.icon className={`size-4 ${pathname === item.url ? "text-primary" : ""}`} />
            <span className="font-medium text-sm">{item.title}</span>
          </Link>
        </SidebarMenuButton>
      </SidebarMenuItem>
    ))}
  </SidebarMenu>
</SidebarContent>
```

Note on SidebarMenuButton height collision: The shadcn sidebar.tsx primitive uses `group-data-[collapsible=icon]:!size-8` with `!important` on collapsed icon state. This means `h-9` only applies in expanded mode — collapsed icon mode uses `size-8` (32px) which is correct and desired. Do not work around this.
